Provides functions for escaping and unescaping HTML text.

Index

  • escape

  • escape

    Function v1.0 
    (escape s)

    Escapes special characters like < to become &lt;. It escapes only five such characters: <, >, &, ' and ".

  • unescape

    Function v1.0 
    (unescape s)

    Unescapes entities like &lt; to become <.
